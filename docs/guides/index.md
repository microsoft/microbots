# Advanced Guides

The Advanced Guides cover reference material and deeper configuration topics for developers who have already completed the **Introduction** and **Getting Started** sections. Use these guides to extend Microbots with custom tools, work with individual bot types, and configure authentication for production deployments.

If you are new to Microbots, start with the [Microbots Introduction](introduction.md), then walk through the **Getting Started** section end-to-end before returning here.
